Markdown 速查

  • 标题：# H1、## H2、### H3 ...
  • 加粗：**bold**；斜体：*italic*
  • 行内代码：`code`；代码块：```lang ... ```
  • 链接：[text](https://example.com)；图片：![alt](url)
  • 列表：- item；数字列表：1. item
  • 引用：> 引用内容
  • 表格：| A | B | + 分隔线与行
  • 水平线：--- 或 ***

使用说明

  • 左侧输入 Markdown，右侧实时预览。
  • 视图选择支持“左右分栏”、“仅编辑”、“仅预览”。
  • 复制/另存为会导出已安全过滤后的 HTML。
  • 输入内容与视图模式会自动保存到本地缓存。
