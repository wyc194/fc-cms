关于正则表达式测试工具

这是一个基于浏览器的在线正则表达式测试工具，所有匹配计算均在本地完成，确保您的数据隐私安全。

主要功能：

  • 实时匹配： 输入正则和文本后立即显示匹配结果。
  • 可视化高亮： 不同颜色的高亮区分交替匹配项，清晰直观。
  • 常用预设： 内置常用的正则表达式（如邮箱、身份证、手机号等），一键应用。
  • 参数控制： 支持全局(g)、忽略大小写(i)、多行(m)等常用修饰符。
