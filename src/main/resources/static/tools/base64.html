关于 Base64 编码转换

Base64 是一种基于 64 个可打印字符来表示二进制数据的表示方法。它常用于在处理文本数据的场合，表示、传输、存储一些二进制数据。

本工具提供以下核心功能：

  • 编码/解码：支持普通文本与 Base64 编码的快速互转，完美支持中文。
  • URL 安全：提供 URL 安全的 Base64 编码（将 + 和 / 替换为 - 和 _），适合在 URL 中传递参数。
  • 文件支持：支持上传图片或其他文件，直接转换为 Data URI (Base64) 格式，方便在 CSS/HTML 中使用。
  • 隐私保护：所有转换逻辑均在浏览器本地执行，您的数据绝不会上传至服务器。
